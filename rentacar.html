"Rentacar" Proyecto FullStack

Proyecto individual basado en el framwork de ReactJs. El objetivo de este ejercicio era de hacer un proyecto FullStack (es decir, con una parte frontEnd y otra Backend) que fueran totalmente funcionales.

Las tecnologías utilizadas son las siguientes:

ReactJs MongoDB MySQL MERN Bootstrap JavaScript HTML CSS JSdoc imaps LocalStorage SessionStorage Responsive

En este proyecto puedo destacar el uso del hook "useParams", que consiste en pasar información a través de la URL, usado en este caso para pasar de un componente a otro la ID del vehículo en cuestión para obtener los sucesivos datos de interes, vease matrícula, número de puertas, etc.
Las bases de datos empleadas son tanto no relacionales (MongoBD) como relacionales (mySQL). Cada una de estas bases de datos almacenan distinta información según la importancia de ésta, es decir, dada la gran cantidad de datos, tipos de archivos y naturaleza de los datos de los vehículos vi conveniente almacenarla en MongoDB, mientras que los datos sensibles y más seguros como son los datos del usuario registrado y logueado van a MySQL.
Esta aplicación web consta de dos partes según el tipo de usuario, cliente y administrador:
La parte de cliente tiene la capacidad de registrarse, loguearse, hacer búsquedas de coches según número de puertas, año de fabricación, marca, país de origen para posteriormente alquilar ese coche en cuestioón, seleccionar las fechas y pagar.
Todo esto en el código se traduce en búsquedas en las dos bases de datos. Con una escritura de datos personales en SQL y otra escritura de fechas en el coche seleccionado.
Parte administrador: Este usuario tiene la potestad de hacer un CRUD completo tanto en SQL para editar, crear y eliminar usuarios. En Mongo DB puede crear, editar y / o eliminar vehículos de alquiler.
Como podemos observar, la parte FrontEnd se ha hecho con Bootstrap-react a pesar de que otorgue un límite en cuanto a personalización, ofrece un diseño totalmente responsive.
