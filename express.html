PLANET EXPRESS!

Tengo el placer de presentaros el proyecto FullBack hecho en grupo con 3 compañeros más. Contiene codigo javascript, páginas en EJS y también en PUG.
En la parte de Backend se ha utilziando tanto bases de datos relacionales como no relacionales (Mongo db Y MySQL) para distintos tipos de datos.
En este primer video muestro la interfaz gráfica y sus funcionalidades.
Las tecnologías utilizadas son las siguientes:

HTML MongoDB MySQL PUG EJS JavaScript CSS JSdoc Leaflet LocalStorage SessionStorage Responsive

En este proyecto puedo destacar de las bases de datos, el objetivo de este ejercicio es hacer CRUD total tanto en Mongo DB como en SQL (bases de datos no relacionales y relacionales.)
Según el tipo de dato irá a SQL o a Mongo DB.
Para hacer código de escritura, lectura, edición y eliminacion de datos se hizo una parte de cliente y otra parte adminstrador, donde el administrador tiene más capacidades que el cliente.
Cabe destacar el uso de la librearía leaflet para el mapa geolocalizado, pudiendo ver la ruta del paquete enviado.

Para más detalles dejo aquí el enlace a GITHUB para poder ver el código en detalles: "Proyecto FullBack EJS PUG".
