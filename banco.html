Formulario bancario algoritmia

Formulario hecho 100%con algoritmia sin funciones ni expresiones regulares. Vemos como en los campos se requiere una forma concreta.
Hemos hecho un resgistro de un banco ficticio con HMTL - CSS - JS.
Las tecnologías utilizadas son las siguientes:

HTML CSS JavaScript

El ejercicio consistía en hacer todo tipo de validaciones que vemos comúnmente en cualquier formulario pero sin el uso de expresiones regulares ni métodos, es decir, hacer funciones para que analicen letra por letra y que estuvieran dentro del marco de condiciones puestas manualmente para cada input.
Por ejemplo, que cada palabra tenga que tener ciertos caracteres que estén dentro de la tabla ASCI, en el correo electrónico contabilizar las letras que tiene que haber después del punto o del @, etc...
Un trabajo puramente aritmético.
